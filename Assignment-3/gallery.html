HTML 5:Figure and Figure Caption

<figure></figure>

Images often come with caption.HTML5 has introduced a new element to contain images and their caption so that the two are associated.
You can have more than one image inside the element as long as they all share the same caption.

<figcaption></figcaption>

the figcaption has been addded to HTML5 in order to allow web pages authors to add a caption to image.
Before thede elements were created there was no way to associated an element with its caption.
Older browsers that do not understand HTML5 elements ignore the new element display the content of them.
